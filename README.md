
# Projeto: Meus Primeiros Passos com HTML e CSS

Este repositório documenta minha jornada de aprendizado com HTML5 e CSS3. Aqui você encontrará exercícios práticos que abrangem desde a estruturação básica de documentos até a estilização visual avançada.

## Sobre o Projeto

O objetivo principal é aplicar os fundamentos do desenvolvimento web front-end. Os exercícios evoluem de tags estruturais básicas para conceitos de design, como teoria das cores e tipografia.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Como Visualizar

Você pode abrir os arquivos `.html` diretamente no seu navegador de preferência.

Para uma experiência de desenvolvimento melhorada, recomenda-se usar a extensão **Live Server** no Visual Studio Code, que atualiza a página automaticamente ao salvar o arquivo.

## Alguns Exercícios

- **ex001** — Meu primeiro site: Uma página simples com um título principal (`<h1>`), parágrafos e uma linha horizontal.
  - **Veja o código:** ex001/index.html
- **ex002** — Parágrafos e quebras de linha: Demonstração do uso da tag `<p>` para parágrafos e `<br>` para quebras de linha forçadas.
  - **Veja o código:** ex002/index.html

- **ex016** — Cores em CSS: Estudo sobre as diferentes formas de representar cores na web, utilizando nomes, códigos hexadecimais e o sistema RGB.
  - **Veja o código:** ex016/cor01.html

- **ex017** — Tipografia e Fontes: Exploração de propriedades de fonte como `font-family`, `font-size` (medidas absolutas e relativas), `font-weight`, e o uso da propriedade shorthand `font`.
  - **Veja o código:** ex017/css/style.css
