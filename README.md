
# Projeto: Exercícios HTML/CSS

Este repositório contém exercícios iniciais de HTML demonstrando estrutura básica de páginas, parágrafos e quebras de linha.

## Como visualizar

Abra os arquivos HTML no seu navegador ou use uma extensão como Live Server no VS Code.


## Exercícios

- **ex001** — Meu primeiro site: uma página simples com `<h1>`, parágrafos e uma linha horizontal. Veja [ex001/index.html](ex001/index.html).
- **ex002** — Parágrafos e quebras de linha: demonstração do uso de `<p>` e `<br>`. Veja [ex002/index.html](ex002/index.html).
